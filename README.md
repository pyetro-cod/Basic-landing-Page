# Landing Page Básica - Estrutura do Projeto

## 📁 Estrutura de Pastas

landing-page/
  public/ (seo & config)
  src/
    html/
    styles/
    scripts/
    assets/
  dist/ (build)
  config files...

---

## 📄 index.html

Estrutura base da página:

- Header (logo + menu)
- Hero section (headline + CTA)
- Sobre
- Serviços / Features
- Depoimentos
- Footer

---

## 🎨 CSS (styles.css)

Organização recomendada:

- Reset / Base
- Layout (grid, flexbox)
- Componentes (botões, cards)
- Responsividade (media queries)

---

## ⚙️ JavaScript (script.js)

Funções básicas:

- Menu mobile toggle
- Scroll suave
- Animações simples (opcional)

---

## 🚀 Boas práticas

- HTML semântico  
- CSS modular  
- Código limpo e comentado  
- Performance (imagens otimizadas)

---

## 📌 Tecnologias sugeridas

- HTML5  
- CSS3 (Flexbox + Grid)  
- JavaScript Vanilla  

---

## 📦 Possíveis melhorias futuras

- Integração com API  
- Formulário com validação  
- SEO básico
